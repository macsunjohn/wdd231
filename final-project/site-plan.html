MovieBox Site Plan

Site Name

MovieBox

This name was selected because it represents a central place where users can explore movies, reviews, and recommendations all in one "box" or platform.

Site Purpose

The MovieBox website provides users with movie recommendations, trending films, reviews, and a platform to explore different genres. It also allows users to discover new movies and save their favourites.

Scenarios

  • What are the top trending movies right now?
  • Where can I find reviews before watching a movie?

Color Scheme

Used for: background, header, footer

Secondary Color (#6A0136)

Used for: buttons, accents, highlights

Typography
